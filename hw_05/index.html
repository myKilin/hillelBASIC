Огляд Селекторів

Селектори бувають

  1. Універсальний селектор приклад

    [*]-Звертається до усіх елементів на сторінці

  2. за назвою тега приклад

    [h1]-Звертається до усіх елементів з такою назвою

  3. по классу

    [.h1]-Звертається до усіх елементів з таким классом

  4. по id

    [#h1]-Звертаеється до тегу id

  5. з декількома классами

    [.h1, .div, .main]-Звертається до усіх классів прописаних через кому

  6. за вкладеністю

    [div ol]-звертається до елемента який прописаний в іншому єлементі
